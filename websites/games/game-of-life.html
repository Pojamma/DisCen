How Conway's Game of Life Works

📋 The Rules

The Game of Life is a cellular automaton where cells evolve based on these four simple rules:

  • Underpopulation: A live cell with fewer than 2 live neighbors dies (as if by loneliness)
  • Survival: A live cell with 2 or 3 live neighbors stays alive to the next generation
  • Overpopulation: A live cell with more than 3 live neighbors dies (as if by overcrowding)
  • Reproduction: A dead cell with exactly 3 live neighbors becomes alive (as if by reproduction)

🎮 How to Use

  • Click cells to toggle them on (alive) or off (dead)
  • Random button generates a random starting pattern
  • Start/Pause controls the simulation
  • Speed slider adjusts how fast generations evolve
  • Clear button resets the entire grid

✨ Interesting Patterns to Try

Click "Random" several times and watch for:

  • Still lifes: Patterns that don't change
  • Oscillators: Patterns that repeat in cycles
  • Gliders: Patterns that move across the grid
  • Spaceships: Larger moving patterns
