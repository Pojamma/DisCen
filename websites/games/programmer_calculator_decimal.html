Number Bases

Click on BIN, DEC, OCT, or HEX to change the number base. Each base allows different digits:

  • Binary (BIN): Only 0 and 1
  • Decimal (DEC): 0-9
  • Octal (OCT): 0-7
  • Hexadecimal (HEX): 0-9 and A-F

Radix Point

The point (.) works in all bases as a radix point. In binary it represents a binary point, in octal an octal point, in decimal a decimal point, and in hexadecimal a hex point. This allows for fractional values in any base.

Bitwise Operations

Bitwise operations work on the binary representation of numbers:

AND (&)

Returns 1 if both bits are 1, otherwise 0.

5 (0101 in binary) & 3 (0011 in binary) = 1 (0001 in binary) Bit comparison: 0101 0011 ---- 0001

OR (|)

Returns 1 if at least one bit is 1.

5 (0101 in binary) | 3 (0011 in binary) = 7 (0111 in binary) Bit comparison: 0101 0011 ---- 0111

XOR (^)

Returns 1 if bits are different, 0 if bits are the same.

5 (0101 in binary) ^ 3 (0011 in binary) = 6 (0110 in binary) Bit comparison: 0101 0011 ---- 0110

NOT (~)

Inverts all bits (0 becomes 1, 1 becomes 0).

~5 (0101 in binary) = -6 Bit inversion: 0101 ---- 1010 (two's complement representation)

Left Shift (<<)

Shifts bits to the left, adding 0s on the right.

5 (0101 in binary) << 2 = 20 (10100 in binary) Bit shift: 0101 << 2 ---- 10100

Right Shift (>>)

Shifts bits to the right, preserving the sign bit.

20 (10100 in binary) >> 2 = 5 (0101 in binary) Bit shift: 10100 >> 2 ---- 00101

Memory Functions

  • M+: Store current value in memory
  • MR: Recall value from memory
  • MC: Clear memory
