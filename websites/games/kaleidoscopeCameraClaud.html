Kaleidoscope App Help

Getting Started

Welcome to the Interactive Kaleidoscope! This app allows you to create beautiful kaleidoscopic patterns using your camera or uploaded images.

Image Sources

  • Use Camera - Click this button to use your device's camera as a live input source. Click again to stop the camera.
  • Upload Image - Select this option to upload any image from your device to use as the source.

Controls

  • Segments - Adjust the number of reflective segments in the kaleidoscope (3-20). More segments create more complex patterns.
  • Zoom - Increase or decrease the zoom level of the source image.
  • Rotation Speed - Control how fast the kaleidoscope rotates. Set to zero for a static image.
  • Color Filter - Apply a color tint to the entire kaleidoscope. This works by using a "multiply" blend mode that combines the original colors with your selected color:
    • White (#ffffff): No change to the original colors
    • Dark colors: Makes the image darker and shifts the color tone
    • Vibrant colors: Creates dramatic colored effects while maintaining details
    • Black (#000000): Would make the entire image black

Interaction

  • Mouse/Touch - Click and drag (or touch and drag) directly on the kaleidoscope to shift the image and create new patterns.
  • Presets - Click on preset buttons below the controls for quick visual styles:
    • Cosmic Spin: Fast rotation with 8 segments
    • Flower: Slow, elegant pattern with 12 segments
    • Star Burst: High zoom with 5 segments
    • Crystal: Fast, intricate pattern with 16 segments

Saving Your Creation

Click the Download Image button to save your current kaleidoscope view as a PNG image file.

Tips

  • Experiment with different segment numbers for dramatically different effects
  • Try using the camera while moving around to create dynamic patterns
  • Use colorful images for the most vibrant results
  • Combine the color filter with different images to create unique moods
